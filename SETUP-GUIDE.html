Partea 1 — Setup inițial (o singură dată)
01
Creează cont GitHub Gratuit
Necesar pentru stocarea fișierelor site-ului

GitHub este locul unde sunt stocate toate fișierele site-ului. Netlify CMS salvează modificările tale (job-uri noi, poze, texte) direct în GitHub, iar Netlify le preia automat și actualizează site-ul.

  1. Mergi la github.com → Sign up
  2. Creează un cont cu email-ul tău
  3. Apasă New repository → numește-l core-recruit
  4. Bifează Public (necesar pentru Netlify CMS gratuit)
  5. Apasă Create repository
Dacă nu vrei să înveți Git, poți folosi GitHub Desktop (aplicație desktop gratuită) pentru a încărca fișierele cu drag & drop.
02
Încarcă fișierele pe GitHub
Upload folderul core-recruit-cms
  1. În repository-ul creat, apasă uploading an existing file
  2. Drag & drop toate fișierele din folderul core-recruit-cms (nu folderul în sine, ci conținutul lui)
  3. Apasă Commit changes

Structura finală pe GitHub trebuie să arate astfel:

core-recruit/ (repository) ├── admin/ │ ├── index.html ← panoul de administrare │ └── config.yml ← configurația CMS ├── content/ │ ├── jobs/ ← fișierele job-urilor │ ├── team/ ← membrii echipei │ ├── testimonials/ ← testimonialele │ └── settings/ ← setările site-ului ├── css/styles.css ├── js/main.js ├── images/uploads/ ← pozele încărcate din CMS ├── index.html ├── about.html ├── careers.html ├── services.html ├── partners.html ├── contact.html └── netlify.toml
03
Conectează Netlify la GitHub Gratuit
Site-ul tău va fi live automat
  1. Mergi la netlify.com → Sign up with GitHub
  2. Apasă Add new site → Import an existing project
  3. Alege GitHub → autorizează accesul → selectează repository-ul core-recruit
  4. Lasă setările de build goale (site static)
  5. Apasă Deploy site

Site-ul tău va fi live în ~30 secunde la o adresă de tip core-recruit-xyz.netlify.app.

Conectare domeniu: Netlify → Site settings → Domain management → Add custom domain → introdu core-recruit.ro și urmează instrucțiunile de DNS.
04
Activează autentificarea CMS Necesar
Securizează panoul de administrare
  1. În Netlify → Site configuration → Identity → Enable Identity
  2. La Registration → schimbă la Invite only (important!)
  3. La External providers → activează GitHub
  4. Mergi la Identity → Git Gateway → Enable Git Gateway
  5. Invită-ți adresa de email: Identity → Invite users → introdu office@core-recruit.ro
Important: Fără pasul "Invite only", oricine poate accesa /admin. Asigură-te că îl bifezi înainte de a face site-ul public.
05
Setează repo-ul în config.yml
Un singur rând de editat

Deschide fișierul admin/config.yml pe GitHub și caută prima linie repo:. Înlocuiește cu username-ul tău GitHub:

backend: name: github repo: username-ul-tau-github/core-recruit ← schimbă asta branch: main

Salvează fișierul (Commit changes) — Netlify va redesploya automat în ~30 secunde.

07
Cum adaugi o poziție nouă
Durează ~2 minute
  1. În panoul admin → click pe Poziții deschise
  2. Click pe + Poziție nouă
  3. Completează formularul:
📋Poziție nouă
  1. Apasă Publică — poziția apare pe site în ~30 secunde!
Tip: Câmpul "Link aplicare" poate fi un URL extern sau un mailto. Recomandat: mailto:office@core-recruit.ro?subject=Aplicatie+Titlu+Pozitie
08
Cum adaugi o fotografie
Poze echipă, logo-uri parteneri

Exemplu: Adaugă fotografia unui membru din echipă

  1. În admin → Echipa → click pe membrul dorit (sau + Membru nou)
  2. La câmpul Fotografie → apasă Choose an image
  3. Apasă Upload → selectează poza de pe calculator
  4. Poza se încarcă automat în folderul /images/uploads/
  5. Apasă Choose selected → poza e setată
  6. Apasă Publică
Formate acceptate: JPG, PNG, WebP, SVG
Dimensiune recomandată: Poze echipă: 800×600px minim · Logo parteneri: format transparent PNG
Dimensiune fișier: sub 2MB pentru viteză optimă
09
Cum editezi textele de pe site
Fără cod, direct din admin

Pentru texte care se schimbă rar (titlu hero, statistici, descrieri servicii):

  1. În admin → Setări site → Setări pagina principală
  2. Editează: titlu hero, descriere, statistici (10+, 200+, 50+)
  3. Apasă Publică

Pentru texte din paginile de servicii, about etc. — acestea se editează direct în fișierele HTML. Dacă nu ești familiarizat cu HTML, poți folosi funcția de search (Ctrl+F) în orice editor text pentru a găsi textul dorit și a-l înlocui.

Editor recomandat pentru fișiere HTML: Visual Studio Code (gratuit) — are highlight de sintaxă și e ușor de folosit.
10
Cum actualizezi site-ul după modificări de cod
Când editezi fișierele HTML direct

Dacă editezi fișierele HTML (texte, structură pagini) direct pe calculator:

  1. Editează fișierul HTML în VS Code sau orice editor text
  2. Mergi pe github.com → repository-ul tău
  3. Click pe fișierul editat → click pe ✏️ Edit (creion) → lipește conținutul nou
  4. Apasă Commit changes
  5. Netlify detectează automat modificarea și redesployează în ~30 secunde
Alternativă mai ușoară: Folosește GitHub Desktop — o aplicație cu interfață grafică unde poți face drag & drop cu fișierele modificate, fără comenzi.
